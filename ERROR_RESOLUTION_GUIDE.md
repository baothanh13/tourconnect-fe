```bash

```

### **2. Restart Development Server**

```bash

cd "c:\Users\LENOVO\tourconnect-fe"

```

### **3. Clear Node.js Cache**

```bash
# If issues persist, clear all caches:

rm -rf node_modules
rm package-lock.json
```
